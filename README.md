###### <h2 align="center"> Arch linux with LUKS encryption </h2>


**Warning**

###### While partitioning with cfdisk, create two partitions only ie, EFI and ROOT. Or make your own tweaks before running it

#####  To install


```
chmod +x basecrypt && ./basecrypt
```
